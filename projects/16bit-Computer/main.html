16-bit Computer

This project is a functional 16-bit computer created using the software Logisim, a program that simulates logic circuits and physical computer architecture. The computer has a dedicated machine code set and a basic programming language, and I'm in the process of writing a compiler in Java to take in a program and compile it into the machine language. This computer supports mathematic operations, variable assignments, basic scripting, function calls, simple objects, and basic recursion.
